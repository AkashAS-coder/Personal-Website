# Profile Website

This is a minimal, professional profile site scaffold.

To preview locally:

```bash
# from project root
python3 -m http.server 8000
# then open http://localhost:8000
```

Files added:
- `index.html` — main page
- `styles.css` — styles and theme
- `script.js` — small interactivity (theme toggle)
- `favicon.svg` — simple logo

Edit text and project entries directly in `index.html`.
